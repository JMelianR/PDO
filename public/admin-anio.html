Cerrar Año Escolar

Esta acción archivará permanentemente toda la información académica del año actual (notas, inscripciones y asignaciones de profesores) y dejará el sistema limpio para el nuevo año escolar. Los usuarios y la estructura de cursos/materias se conservan.

📊 Todas las notas se archivarán y se eliminarán del sistema activo.
👥 Todas las inscripciones de alumnos a cursos se limpiarán.
👨‍🏫 Todas las asignaciones de profesores a cursos se desvinculan.
🔒 Esta operación no se puede deshacer. Los datos quedan en el historial.
